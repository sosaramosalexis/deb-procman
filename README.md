# Deb ProcMan — Process Manager

Interactive whiptail-based process and service manager for Debian/Ubuntu.

## Quick Start

```bash
su -
bash <(curl -fsSL https://raw.githubusercontent.com/sosaramosalexis/deb-procman/main/install.sh)
```

## Features

- **Process list** — view running processes with CPU/memory usage, search/filter
- **Service list** — browse systemd services with active/enabled status, search/filter
- **Favorites** — bookmark frequently used processes and services for quick access
- **Actions** — start, stop, restart, kill, enable/disable at boot (bootstart)

## Usage

1. **Running processes** — browse by CPU usage, search by name, stop/kill or toggle favorite
2. **Systemd services** — browse all services, search, start/stop/restart, enable/disable at boot
3. **Favorites** — quick access to your bookmarked processes and services
